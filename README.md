# USP.ai Smart Website Growth Engine

This project is an interactive landing page and sales tool designed to demonstrate the value of the USP.ai "Smart Website" system. It serves as a high-performance guide that contrasts traditional "brochure" websites with modern, integrated growth engines.

## üöÄ Overview

Built with modern web technologies, this application is designed to be deployed as a subdomain (e.g., `growth.usp.ai`) to complement the main WordPress site. It features:

- **High-Performance Rendering:** fast load times and smooth transitions.
- **Scroll-Triggered Animations:** Visual storytelling using Framer Motion.
- **Interactive Mockups:** Live demonstrations of mobile responsiveness, CRM pipelines, and unified inboxes.
- **Data Visualization:** Animated charts showing ROI and analytics.
