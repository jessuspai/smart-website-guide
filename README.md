# USP.ai Smart Website Growth Engine

An interactive, high-performance sales guide for the USP.ai Smart Website Growth System. Built with React, Tailwind CSS, and Framer Motion.

This is the interactive sales guide for USP.ai.

## Project Overview
This landing page demonstrates the difference between a standard "brochure" website and the USP.ai "Smart System". It uses high-performance code (React) to show animations and interactive demos.

## How to Run
1. Open this folder in the terminal.
2. Run `npm install` to get the tools.
3. Run `npm run dev` to start the website on your computer.

## Deployment
This project is ready to upload to Vercel.com to create a live link (e.g., growth.usp.ai).
